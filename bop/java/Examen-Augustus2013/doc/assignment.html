Jaarprogramma

Opgave

Studenten stellen bij aanvang van en tijdens het nieuw academiejaar 2013-2014 hun jaarprogramma op. We wensen een informatiesysteem te ontwikkelen voor het opstellen van de jaarprogrammaâ€™s van studenten voor het academiejaar 2013-2014.

Elke student heeft een studentnaam, studentvoornaam en studentcode. De studentcode is een positief geheel getal, bestaande uit 6 cijfers. Het jaarprogramma van een student bestaat uit een rij van inschrijvingen. Elke inschrijving betreft het opnemen van één opleidingsonderdeel (OPO) op een bepaalde datum. Wanneer een student zich inschrijft voor een OPO wordt dus ook bijgehouden op welke datum dat deze inschrijving gebeurt. Inschrijvingen kunnen enkel gebeuren tussen 1 september en 1 april. Je mag er van uitgaan dat het aantal inschrijvingen in een jaarprogramma van een student maximaal gelijk is aan 20.

Een OPO heeft een code, een naam en een aantal studiepunten. Verder wordt ook bijgehouden of het OPO al dan niet in het Engels wordt gegeven. Je kan je uiteraard geen tweemaal inschrijven voor hetzelfde OPO (d.i. twee OPO's met eenzelfde code).

  • Voorzie meerdere constructoren:
    • Een OPO wordt standaard niet in het Engels gegeven.
    • Een OPO heeft standaard 3 studiepunten.
  • Het minimum aantal studiepunten voor een OPO bedraagt 1, het maximum 20.
  • De methode zelfdeAls dient om OPO's te vergelijken. Twee OPO's zijn gelijk indien ze dezelfde code hebben. Andere velden worden genegeerd.
  • De methode format moet het OPO-object omzetten naar een String. Bijvoorbeeld: Anatomie, code = 1234, gegeven in het Engels, 10 stp

Implementatie Jaarprogramma

Schrijf de code voor de klasse Jaarprogramma en doe de tests werken. Volgende methodes moeten bestaan:

  • void voegInschrijvingToe(Inschrijving inschrijving)
