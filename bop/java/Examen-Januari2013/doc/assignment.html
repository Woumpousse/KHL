Overzicht

Voor deze opgave moet je een programma schrijven voor een kinderdagverblijf. Het moet mogelijk zijn om kinderen in te schrijven op bepaalde dagen en uit te rekenen hoeveel gefactureerd moet worden. We bouwen de functionaliteit stapsgewijs op.

Je krijgt van ons een klasse Datum; deze bevat belangrijke functionaliteit waar je zal op moeten steunen om deze opgave uit te werken.

Kind

We beginnen met een klasse voor het voorstellen van kinderen.

Een kind heeft de volgende kenmerken; vul bij elk het type in:

  • Voornaam ? Een voornaam kan het best voorgesteld worden d.m.v. een String.
  • Familienaam ? Een familienaam kan het best voorgesteld worden d.m.v. een String.
  • Geboortedatum ? De meegeleverde Datum klasse kan hier van pas komen.
  • Leeftijd ? Een double zou hier kunnen werken, maar we kiezen hier voor een int.

Hoewel we hier vier kenmerken opsomden, zullen we slechts drie velden nodig hebben vermits één dan de kenmerken berekenbaar is. Welke?

Fout...

Kan men iemand zijn geboortedatum afleiden uit zijn leeftijd?

Juist: gegeven de geboortedatum kan men de leeftijd uitrekenen.

Schrijf de klasse Datum zodat de tests slagen.

  • De methode die de leeftijd opvraagt, verwacht een Datum-argument: de methode geeft dan terug welke leeftijd het kind heeft op die gegeven datum.
  • Hou de setters private.
  • Check overal op ongeldige argumenten. Gebruik throw new IllegalArgumentException(); bij foute invoer.

Inschrijving

Het is mogelijk om een kind in te schrijven in het kinderdagverblijf voor een specifieke dag. We kunnen daarbij kiezen voor opvang in de voormiddag, namiddag of beide. Een inschrijving kost geld: €29.95 indien het om een halve dag gaat, €49.95 voor een volledige dag.

Vul de types in voor de volgende kenmerken van een inschrijving:

  • Kind ? We herbruiken hier de klasse Kind die we daarnet gemaakt hebben.
  • Datum ? Datum.
  • Voormiddag en/of namiddag ? Een boolean is de meest voor de hand liggende manier om aan te geven of we het kind al dan niet inschrijven voor de voor- of namiddag.
  • Kost ? Het gaaat om kommagetallen, dus we kiezen voor double.

Schrijf de klasse Inschrijving zodat de tests slagen.

  • Hou de setters private.
  • Check overal op ongeldige argumenten. Gebruik throw new IllegalArgumentException(); bij foute invoer.
