Recursion

Write a recursive function isOdd(n) that checks whether the given integer n is odd or not. An odd number is one that is not divisible by 2.

Write a recursive function factorial(n) that computes the factorial of its argument n. The factorial of n is defined as 1 * 2 * ... * n.

Write a recursive function sum(xs) that computes the sum of the numbers in the array xs.

Write a recursive function product(xs) that computes the product of the numbers in the array xs.

Write a recursive function countZeros(xs) that counts the number of zeros appearing in the array xs.

Write a recursive function removeZeros(xs) that returns a new array equal to the given array xs but with all zeros removed.

Write a recursive function firstIndexOf(x, xs) that looks for the value x in the array xs and returns its index within that array. If the value x does not appear in the array xs, -1 is returned.

Write a recursive function removeAt(xs, index) that returns a new array equal to the given array xs but with the element at the given index removed.

Write a recursive function isSubsetOf(xs, ys) that checks whether all elements of xs appear at least as often in ys.

Write a recursive function range(a, b) that returns an array containing the numbers a until (and including) b.

Write a recursive function duplicateEachItem(xs) that creates a new array where each item of xs appears twice in a row.

Write a recursive function subarrays(xs) that lists all possible subarrays of xs. An array A is a subarray of an array B if each item from A appears in B, in the same order.

Write a recursive function permutations(xs) that creates a new array containing all permutations of xs. An array A is a permutation of an array B if A and B contain the same items, but not necessarily in the same order.

Write a recursive function knapsack(n, ns) that looks for a subarray of ns such that its sum equals n. If no such subarrays exists, null must be returned.
