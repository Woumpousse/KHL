2D Arrays

Note: all grids must be represented as arrays of columns, not arrays of rows.

Write a function createGrid(width, height, value) that creates a 2D-array representing a grid with dimensions width by height. Each cell in the grid must contain the value value.

Write a function arrayLengths(xss) that, given an array of arrays xss, determines the length of each array in xss.

An array can contain other arrays. If all these arrays have the same length, it is easy to interpret it as a grid or matrix. Write a function isRectangular(xss) that, given an array of arrays xss, checks whether all arrays inside xss have the same length.

Write a function width(xss) that returns the width of the grid xss. You can assume that xss is rectangular.

Write a function height(xss) that returns the height of the grid xss. You can assume that xss is rectangular.

Write a function isSquare(xss) that determines whether xss represents a square matrix, i.e. that its width and height are equal.

Write a function zigZag(width, height) that returns a grid with size width by height and containing numbers in zig-zag pattern.

Write a function getRow(xss, row) that returns a copy of the row-th row of the grid xss.

Write a function getRow(xss, col) that returns a copy of the col-th column of the grid xss.

Write a function rowSums(xss) that returns an array whose i-th item corresponds to the sum of the values of the i-th row of xss.

Write a function columnSums(xss) that returns an array whose i-th item corresponds to the sum of the values of the i-th column of xss.
