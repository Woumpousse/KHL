Conditionals

Write a function average(a, b) that computes the average of a and b. The average of two numbers is equal to half their sum.

A book costs €5. You buy n books. Write a function cost1(n) that computes how much you have to pay.

A book costs €5, a DVD costs €7. You buy n books and k DVDs. Write a function cost2(n, k) that computes how much you have to pay.

A book costs €5, a DVD costs €7. You buy n books and k DVDs. There is a shipping cost of €0.5 per item. Write a function cost3(n, k) that computes how much you have to pay.

A book costs €5, a DVD costs €7. You buy n books and k DVDs. There is a shipping cost of €0.5 per item. You only need to pay shipping costs if the total item cost (i.e. without shipping and handling) is less than €20. Write a function cost4(n, k) that computes how much you have to pay.

A book costs €5, a DVD costs €7. You buy n books and k DVDs. There is a shipping cost of €0.5 per item. You only need to pay shipping costs if the total item cost (i.e. without shipping and handling) is less than €20 or if you bought less than 3 items. Write a function cost5(n, k) that computes how much you have to pay.

Write a function isOdd(n) that determines whether n is odd. A number is odd if it is not divisible by two.

Write a function idEven(n) that determines whether n is even. A number is even if it is divisible by two.

Write a function endsInZero(n) that determines whether the given number n ends with a zero.

Write a function endsInThreeZeros(n) that determines whether the given number n ends with three zeros.

Write a function minimumOfTwo(a, b) that finds the least of the given numbers a and b.

Write a function abs(a) that computes the absolute value of a. The absolute value of a number a is the number itself if a is positive, or its negation if its is negative.

Write a function maximumOfTwo(a, b, c) that finds the greatest of the given numbers a, b and c.

Write a function sort(a, b, c) that orders the three given values a, b and c.
