Arrays

Schrijf een functie contains(xs, x) die nagaat of een gegeven waarde x voorkomt in de array xs.

Schrijf een functie sum(xs) die de som van alle getallen in de array xs uitrekent.

Schrijf een functie product(xs) die het product van alle getallen in de array xs uitrekent.

Schrijf een functie average(xs) die het gemiddelde uitrekent van alle waarden in de array xs.

Schrijf een functie incrementEachItem(xs) die 1 toevoegt aan elk getal in de array xs.

Schrijf een functie countZeros(xs) die telt hoeveel nullen er voorkomen in de array xs.

Schrijf een functie firstIndexOf(x, xs) die zoekt naar het eerste voorkomen van x in de array xs en de index ervan teruggeeft. Indien x niet voorkomt in xs, moet het resultaat -1 zijn.

Schrijf een functie lastIndexOf(x, xs) die zoekt naar het laatste voorkomen van x in de array xs en de index ervan teruggeeft. Indien x niet voorkomt in xs, moet het resultaat -1 zijn.

Schrijf een functie range(a, b) die een array opstelt met erin de getallen a, a+1, ..., b.

Schrijf een functie reverse(xs) die de gegeven array xs omkeert. De array xs moet hiervoor gewijzigd worden; m.a.w. steek het resultaat niet in een nieuwe array.

Schrijf een functie isPalindrome(xs) die nagaat of de array xs een palindroom is, d.i. de array is gelijk aan zijn omgekeerde. Bijvoorbeeld, [1,2,3,2,1] is een palindroom.

Schrijf een functie minimum(xs) die het kleinste getal uit de array xs opzoekt.

Schrijf een functie maximum(xs) die het grootste getal uit de array xs opzoekt.

Schrijf een functie isIncreasing(xs) die nagaat of de getallen in de array xs stijgend zijn. Gelijke elementen mogen voorkomen; m.a.w. [1,2,2,3] is een stijgende array.

Schrijf een functie isDecreasing(xs) die nagaat of de getallen in de array xs dalend zijn. Gelijke elementen mogen voorkomen; m.a.w. [3,2,1,1] is een dalende array.

Schrijf een functie areEqual(xs, ys) die nagaat of de arrays xs en ys dezelfde elementen in dezelfde volgorde bevatten.

Schrijf een functie alternates(xs) that partitions the given array xs in two arrays, where the first contains all of xs's items with even indices and the second all of xs's items with odd indices.
