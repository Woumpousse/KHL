Arrays

Write a function contains(xs, x) that checks whether the given value x appears somewhere in the array xs.

Write a function sum(xs) that computes the sum of all integers in the array xs.

Write a function product(xs) that computes the product of all integers in the array xs.

Write a function incrementEachItem(xs) that adds one to each value in the array xs.

Write a function countZeros(xs) that returns the number of zeros appearing in the array xs.

Write a function firstIndexOf(x, xs) that looks for the first appearance of x in the array xs and returns its index. Return -1 if x does not appear in xs.

Write a function lastIndexOf(x, xs) that looks for the last appearance of x in the array xs and returns its index. Return -1 if x does not appear in xs.

Write a function range(a, b) that constructs an array containing the numbers a, a+1, ..., b.

Write a function reverse(xs) that modifies the given array xs so that its contains its items in reverse order.
