2D Arrays

Opmerking: alle 2D-arrays moeten bijgehouden worden als een lijst van kolommen, niet als een lijst van rijen.

Schrijf een functie arrayLengths(xss) die, gegeven een array van arrays xss, de lengte van elk van de arrays teruggeeft in een array. Bijvoorbeeld, de array [ [1], [1,2], [1,2,3] ] bevat drie arrays. De eerste heeft lengte 1, de tweede lengte 2 en de derde lengte 3. Deze drie lengtes moeten teruggegeven worden als een nieuwe array: [1,2,3].

Een array kan zelf andere arrays bevatten. Indien deze arrays allemaal dezelfde lengte hebben, kan met de array-van-arrays interpreteren als een rooster of een matrix. Schrijf de functie isRectangular(xss) die, gegeven een array van arrays xss, nagaat of elke array dezelfde lengte heeft.

Schrijf een functie createGrid(width, height, value) die een 2D-array aanmaakt met breedte width en hoogte height. Elk element moet geïnitialiseerd worden op value.

Schrijf een functie width(xss) die gegeven een 2D-array xss de breedte ervan teruggeeft.

Schrijf een functie height(xss) die gegeven een 2D-array xss de hoogte ervan teruggeeft.

Schrijf een functie isSquare(xss) die nagaat of de gegeven 2D-array xss vierkant is.

Schrijf een functie zigZag(width, height) die een 2D-array teruggeeft met de gegeven dimensies. De waarden erin moeten een zigzag vorm maken, zoals hieronder getoond wordt.

Schrijf een functie getRow(xss, row) die gegeven een 2D-array xss een kopie geeft van de rij met index row.

Schrijf een functie getColumn(xss, column) die gegeven een 2D-array xss een kopie geeft van de kolom met index column.

Schrijf een functie rowSums(xss) die gegeven een 2D array xss voor elke rij de som uitrekent en deze in een nieuwe array plaatst.

Schrijf een functie columnSums(xss) die gegeven een 2D array xss voor elke kolom de som uitrekent en deze in een nieuwe array plaatst.

Schrijf een functie matrixAddition(xss, yss) die gegeven een matrices xss en yss de som van beide matrices uitrekent.

Schrijf een functie matrixMultiplication(xss, yss) die gegeven een matrices xss en yss het product van beide matrices uitrekent.
