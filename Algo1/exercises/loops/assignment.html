Arrays

Write a function rangeSum(a, b) that computes the sum of all numbers from a to b: a + (a + 1) + ... + b.

Write a function countTrailingZeros(n) that counts the number of trailing zeros in the decimal notation of n. For example, 1000 has three trailing zeros.

Write a function countZeros(n) that counts the number of zeros in the decimal notation of n. For example, 1010 has two zeros.

Write a function isPrime(n) that determines whether n is prime. A number is prime if it is divisible only by 1 and itself. 1 is not considered a prime.
