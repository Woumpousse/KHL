Loops

Write a function rangeSum(a, b) that computes the sum of all numbers from a to b: a + (a + 1) + ... + b.

Write a function factorial(n) that computes the factorial of n, which is equal to 1 * 2 * 3 * ... * n.

Write a function gcd(a, b) that computes the greatest common divisor of a and b. The greatest common divisor k of two numbers a and b is the greatest number such that a % k === 0 and b % k === 0.

Write a function lcm(a, b) that computes the least common multiple of a and b. The least common multiple k of two numbers a and b is the least number such that k % a === 0 and k % b === 0.

Write a function countDigits(n) that counts the number of digits in the decimal notation of n. For example, 12345 has 5 digits.

Write a function countTrailingZeros(n) that counts the number of trailing zeros in the decimal notation of n. For example, 1000 has three trailing zeros.

Write a function countZeros(n) that counts the number of zeros in the decimal notation of n. For example, 1010 has two zeros.

Write a function isPrime(n) that determines whether n is prime. A number is prime if it is divisible only by 1 and itself. 1 is not considered a prime.
